# 🗨 KKOTalk
<br/>

## 📄 Description

- 닉네임을 설정하여 익명의 사람들과 대화 할 수 있는 **실시간 오픈 채팅 서비스** 입니다.
- 실시간으로 **이용자의 수**를 알 수 있으며 입장과 퇴장시 안내 메세지를 띄웁니다.
- 채팅 메세지가 도착한 **시간**을 확인할 수 있습니다.
<br/>

## 🌏 Browser Support

- Chrome (latest version)
- Microsoft Edge (latest version)
<br/>

## 🛠 Skills

- Node.js, Express
- Socket.io
- AWS EC2
